# 药物信息学研究项目

本项目用于整理和管理药物信息学研究资料，特别关注乙肝药物研发相关内容。

## 项目结构

```
pharmacoinformatics/
├── literature/          # 文献资料
│   - 存放相关研究论文
│   - 文献综述
│   - 重要参考资料
│
├── drug_discovery/      # 药物发现
│   └── hbv/            # 乙肝相关研究
│       - 作用机制研究
│       - 药物靶点分析
│       - 候选药物筛选
│
├── methods/            # 研究方法
│   - 计算机辅助药物设计
│   - 分子对接方法
│   - 机器学习应用
│
├── data/              # 数据集
│   - 实验数据
│   - 分析结果
│   - 数据可视化
│
└── notes/             # 学习笔记
    - 每日研究记录
    - 重要概念整理
    - 技术要点总结

## 使用指南

1. 文献整理
   - 将PDF格式的文献存放在 `literature` 目录
   - 使用markdown格式记录文献笔记

2. 研究记录
   - 在 `notes` 目录下使用markdown格式记录日常研究进展
   - 重要的研究发现及时更新到相应的目录中

3. 数据管理
   - 原始数据存放在 `data` 目录
   - 保持良好的数据组织结构和命名规范

## 注意事项

1. 定期备份重要数据
2. 保持目录结构整洁
3. 及时更新研究进展
4. 遵循数据管理规范
