# 乙肝药物研发学习路线图

## 概述
本文档概述了结合AI进行乙肝药物研发所需的核心知识体系和学习路线。

## 知识领域优先级

1. 乙肝病毒学
2. 数据科学与AI应用
3. 药物化学与分子设计
4. 免疫学与免疫调节
5. 临床前研究与实验设计
6. 伦理学与法规

## 学习建议

1. 循序渐进：先掌握基础病毒学知识，再学习技术工具
2. 理论结合实践：在学习AI工具的同时，进行实际的数据分析练习
3. 持续更新：保持对最新研究进展的关注
4. 跨学科学习：注重多个领域知识的融会贯通
